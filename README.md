# streamlit_sample

streamlit sample project

# Installation

1. downlaod data

```bash
cd data
./download.sh
```

2. download elasticsearch plugin 

```bash
cd env/elasticsearch
./download.sh
```

# Usage

```bash
docker-compose up
```
